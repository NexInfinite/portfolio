WHYTHON 

Whython is an almost fully customizable language written in python, its syntax is based of Visual Basic. This is a project language written for fun which allows you to create your own language; from pseudocode syntax to emoji code, the choice is yours.

VIDEO TO ASCII 

Video to ASCII was a project written over a weekend which takes a YouTube video and converts the output the ASCII art in a bash terminal in real time. It was written in Python and had some updates later on which pushed real time rendering of your monitor.

AWARDS AND COMPETITIONS

Take a look at competitions I've attended and awards I have won

SHEFJAM 9 WINNERS 

Shefjam is a competition ran by the Sheffield Computer Science Society. Its a 42 hour programming a event where we have to make a game based on a theme, this year being villains. Our team developed a PvE game where the player has to survive 5 Nights in Newcastle. One player tries to survive and another controls the environment to stop them from surviving. We won best game overall and best name.

PROJECT FALCON 

Project Falcon is a Sheffield student lead project for IMechE. This is a competition where students have to build and fly an autonomous drone through a track. We will be competing in IMechE 2023 and I will be responsible for helping create the autonomous side of the drone. I am part of the missions engineering team, which is a team consisting of 5 Sheffield students from many different courses.
